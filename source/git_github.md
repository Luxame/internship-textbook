## 練習題一

1. 開一個分支 intern/your_id
2. 做一點修正
3. 發 pull request
4. pull request 對象為 mater

## 練習題二

1. fork 回去自己的 github
2. 新增一個自己 id 的 md 黨
3. 發 pull request 要求 merge 到 https://github.com/trunk-studio/sandbox 之 master branch
