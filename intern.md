# 實習計畫

創科資訊位於台中市中心，是一家年輕有活力的新創公司，秉持對軟體設計的專業與熱情，擁有敏捷軟體開發團隊及高效率的工作流程，專注於提供高品質軟體研發及培訓服務，合作夥伴及客戶來自歐美及兩岸三地。

## 主要產品

數位內容、行動電商及物聯網應用軟體研發及培訓服務。

## 發展願景

創科資訊致力於打造軟體人才所追求理想工作，特別重視優秀軟體人才的養成，並且不斷追求更好的軟體開發觀念和技術，長程目標是發展成為世界級的軟體研發團隊。

## 實習職位

主要工作內容：參與軟體開發團隊。

基本要求：程式設計實作基礎能力

* Web Designer

  技能需求：熟悉 HTML5 / CSS3 / JavaScript 等現代網頁開發技術。

* Software Developer

  技能需求：熟悉前端或後端軟體開發，以 Java 或 JavaScript 為主。

## 工作地點

台中市西區台灣大道二段二號16F-1

## 實習方式

* 教育部大學生校外實習計畫

  配合學校作業核發正式學分，依照合約期間，可分為暑期、學期與學年度實習，依規定發給獎助學金或工作薪資。

* 外包合作方式

  簽訂專案實習聘僱合約，依照完成交付的開發工作給予獎助學金或工作薪資。

## 工作福利

彈性工作時間制度；結案獎金；聚餐；免費咖啡飲料暢飲；專業培訓課程
